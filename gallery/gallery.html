BeeSnap App (01/2017 - Present)

Purpose: The purpose of this project is to design a web application for K-12 Students to classify types of bees for educational purposes. The intention is that the app will include a dichotomous key that will make it easier for students to identify the bees that they encounter. Ultimately, students will be able to take pictures of bees that will be geotagged and sent along with information from the dichotomous key to our database. Since the targeted audience is younger school children, the UI of the application is very streamlined and minimalistic to allow for easy, focused and intuitive user experience.

Design: This web app is being built in Angular 1.0. AngularJS is the web application framework we are using in order to allow for easier front-end and back-end integration. The group works in a collaborative Agile method in designing the app.

Spring 2017: I worked on the CS team (database-subteam). I created the relevant tables and added in the dichotomous key data (stored as bitstrings) into the tables. I assisted team members in debugging the integration of the camera functionality into the app.

GBA-Game (10/2016 - 12/2016)

I worked on creating a Game Boy Advance game coded in C using standard libraries. A .gba file is available on my github. The game can be played using any standard GBA emulator.

Game Description:
PUT OUT THE FIRE!

The fire department needs your help! They need a way to be guided to all of the wild fires happening around town. Use your Fire-Proof dog to guide them to all the fires by chasing the arson's mark: cookies!

Gameplay:
* Eat all the cookies using your square player and avoid fires!
* Eat 10 cookies and YOU WIN!
* Press A to speed up & B to slow down again! (Very difficult to control)
* Scores start from 1 because everyone is a winner!
